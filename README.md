# Mendix AI Suite

Enterprise BDR tool voor Mendix prospecting — gebouwd met Claude AI.

## Features
- Multi-account management
- Research, Signals, Account Map agents
- Contactbeheer met sjabloon-flow
- Sequence builder met outreach-tracking
- BDR, Email Reply, Battle Card, Objections, Pipeline agents
- Token usage tracking
- Supabase backend (binnenkort)

## Deployment

### Automatisch (aanbevolen)
1. Pas `mendix-suite.html` aan
2. Voer `push.bat` uit
3. GitHub Actions deployt automatisch naar GitHub Pages

### GitHub Secrets vereist
| Secret | Beschrijving |
|--------|-------------|
| `APP_SUPABASE_URL` | Supabase project URL |
| `APP_SUPABASE_KEY` | Supabase anon/public key |
